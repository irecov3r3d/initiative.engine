# Security Policy

## Supported Versions

Please see the table below for supported versions and their security update status.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly.

Instead, please report the vulnerability by emailing the maintainers directly at security@initiative.engine. We will strive to acknowledge your report within 48 hours and will provide a timeline for resolution.
